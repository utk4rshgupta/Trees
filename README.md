# Trees Project

This is a Java project that implements various tree data structures, including a basic Binary Tree and a Binary Search Tree (BST). The project demonstrates tree construction, traversal, and visualization.

## Files

### BinaryTree.java
This file contains the `BinaryTree` class, which implements a basic binary tree data structure.

**Key Features:**
- `populate(Scanner scanner)`: Interactively builds the tree by prompting the user for node values and their children.
- `display()`: Prints the tree in a simple indented format.
- `prettyDisplay()`: Displays the tree in a visually appealing tree-like structure using ASCII art.

**Usage:**

### BST.java
This file contains the `BST` class, which implements a Binary Search Tree with additional features like balancing checks and traversals.

**Key Features:**
- `insert(int value)`: Inserts a value into the BST while maintaining the BST property.
- `populate(int[] nums)`: Inserts all values from an array into the BST.
- `populateSorted(int[] nums)`: Builds a balanced BST from a sorted array.
- `balanced(Node node)`: Checks if the subtree rooted at the given node is balanced.
- `display()`: Prints the tree with details about each node's children.
- `prettyDisplay()`: Displays the tree in a tree-like structure using Unicode box-drawing characters.
- Traversal methods: `preOrder()`, `postOrder()`, `inOrder()` for different tree traversals.

**Usage:**
- Run the `main` method to create a BST from an array and display it.

### AVL.java
This file contains the `AVL` class, which implements an AVL (Adelson-Velsky and Landis) self-balancing binary search tree.

**Key Features:**
- `insert(int value)`: Inserts a value into the AVL tree while maintaining balance through rotations.
- `display()`: Prints the tree structure in an indented format.
- Left and right rotation operations to maintain balance after insertions.
- Height tracking for each node to determine when rebalancing is needed.
- Balance factor calculation to determine the balance state of nodes.

**Usage:**
- Create an AVL tree instance and use the `insert()` method to add values.
- The tree automatically rebalances itself to maintain O(log n) height.

### SegmentTree.java
This file contains the `SegmentTree` class, which implements a Segment Tree data structure for efficient range sum queries and point updates.

**Key Features:**
- `constructTree(int[] arr, int start, int end)`: Builds the segment tree from an array using a bottom-up approach.
- `query(int qsi, int qei)`: Returns the sum of elements in the range [qsi, qei] in O(log n) time.
- `update(int index, int value)`: Updates a single element at the given index and updates all affected parent nodes in O(log n) time.
- `display()`: Visualizes the tree structure showing intervals and data values.

**Usage:**
- Create a SegmentTree with an array: `SegmentTree tree = new SegmentTree(arr);`
- Query range sum: `tree.query(start, end);`
- Update a value: `tree.update(index, newValue);`

**Time Complexity:**
- Construction: O(n)
- Query: O(log n)
- Update: O(log n)
