# Trees Project

This is a Java project that implements various tree data structures, including a basic Binary Tree and a Binary Search Tree (BST). The project demonstrates tree construction, traversal, and visualization.

## Files

### BinaryTree.java
This file contains the `BinaryTree` class, which implements a basic binary tree data structure.

**Key Features:**
- `populate(Scanner scanner)`: Interactively builds the tree by prompting the user for node values and their children.
- `display()`: Prints the tree in a simple indented format.
- `prettyDisplay()`: Displays the tree in a visually appealing tree-like structure using ASCII art.

**Usage:**
- Run the `main` method to create and display a binary tree interactively.

### BST.java
This file contains the `BST` class, which implements a Binary Search Tree with additional features like balancing checks and traversals.

**Key Features:**
- `insert(int value)`: Inserts a value into the BST while maintaining the BST property.
- `populate(int[] nums)`: Inserts all values from an array into the BST.
- `populateSorted(int[] nums)`: Builds a balanced BST from a sorted array.
- `balanced(Node node)`: Checks if the subtree rooted at the given node is balanced.
- `display()`: Prints the tree with details about each node's children.
- `prettyDisplay()`: Displays the tree in a tree-like structure using Unicode box-drawing characters.
- Traversal methods: `preOrder()`, `postOrder()`, `inOrder()` for different tree traversals.

**Usage:**
- Run the `main` method to create a BST from an array and display it.
